Tennis

Le tennis est un sport de raquette qui se joue entre deux joueurs (ou deux paires de joueurs dans le cas du double). L'objectif du jeu est de frapper une balle en caoutchouc creuse recouverte de feutre sur le court de l'adversaire de manière à ce qu'il ne puisse pas la renvoyer. Le joueur qui remporte le plus de points dans un set gagne le set. Le joueur qui remporte le plus de sets dans un match gagne le match.

Le tennis est un sport très populaire dans le monde entier. Il est joué par des personnes de tous âges et de tous niveaux. C'est un sport idéal pour rester en forme et s'amuser.
